# هدف پروژه
هدف این پروژه آموزش مدلیست که بتواند اعداد دستنویس فارسی
و عملگر های ریاضی را از روی تصویر تشخیص دهد.

# چالش پروژه
چالشی که وجود دارد این است که تصاویر درنظر گرفته شده برای 
آموزش مدل در واقع بعد از تبدیل شدن از فرمت CSV،
28×28 هستند و تصاویر درنظر گرفته شده برای تست نهایی
336x28 هستند.درواقع 12 تصویر 28×28 در کنارهم قرار گرفته و
برای این‌ امر باید الگوریتمی نوشته شود که این تصاویر یک به یک و
به ترتیب به مدل داده شود برای پیشبینی نهایی.

# نحوه عملکرد مدل ها
برای این پروژه دو مدل نظر گرفته شده که یکی از مدل ها روی
داده اصلی آموزش دیده تا اعداد دستنویس را حدس بزند و مدل
دیگر عملگر ها و تصاویر پوچ را بتواند پیشبینی کند.درواقع مدل 
دوم روی مدل اول و با استفاده از داده هایی که از دیتاست اصلی
تهیه شده Finetune شده است تا بتواند به نوعی پیشبینی مدل
اول را به روز کند.
در معماری هر دو مدل از CNN استفاده شده است.
انتهای هر فایل گزارش مربوط به Epoch نهایی قرار داده شده است.

# کتابخانه های مورد نیاز
برای این پروژه از کتابخانه های زیر استفاده شده است:
opencv-pandas-matplotlib-tqdm-torch-
torchmetrics

# نکته پایانی
به علت حجیم بودن فایل های مربوط به آموزش و ارزیابی،فقط
دیتاست مربوط به Finetuning قرار داده شده که همانطور که
گفته شد از تصاویر اصلی دیتاست گرفته شده و تصاویر اصلی
نیز دقیقا به همین صورت ذخیره شده اند.مراحل پیش پردازش
فاز آموزش و Finetuning به یک صورت هستند.
وزن های مدل آموزش دیده و Finetune شده قرار داده شده است.
